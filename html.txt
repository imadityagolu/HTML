//HTML

HyperText Markup Language, It's a markup language used to create and give structure to a web page.
HTML is the foundation of the web, and it's used in conjunction with CSS and JavaScript to build websites.
The web pages can be static web pages or dynamic web pages.
It is also used to make web applications.
It helps users navigate the internet.


// its purpose, importance in web development, and a brief history.

Web development focuses on creating and maintaining websites or applications, 
integrating front-end and back-end technologies for functionality and usability. 
It drives global communication, commerce, and accessibility, forming the backbone of the digital world. 
Originating in 1989 with Tim Berners-Lee's World Wide Web, 
it evolved from static HTML pages to dynamic, interactive platforms powered by technologies 
like JavaScript, CSS, and modern frameworks like React, enabling sophisticated and responsive user experiences.


//inline elements
Inline elements occupy only the width of their content and do not start on a new line.
They flow within a block-level container and align with neighbouring content.
Examples: <span>, <a>, <strong>, <em>, <img>.
--explaining one example ie; span
The <span> remains within the same line as the surrounding text and takes only as much space as its content.


//block-line elements
Block-level elements take up the full width of their parent container and start on a new line.
They stack vertically and can contain other block or inline elements.
Examples: <div>, <p>, <h1>-<h6>, <ul>, <section>.
--explaining one example ie; div
The <div> spans the full width of its container, even if the content inside is smaller.


//FORM
HTML forms are a collection of elements used to collect user input and submit it to a server for processing. 
They are defined using the <form> tag and typically include input elements 
such as text fields, checkboxes, radio buttons, dropdowns, and submit buttons.

The <input> element is used to create various types of input fields like text, email, password, etc., controlled by the type attribute.
The <textarea> element allows users to input multi-line text.
The <select> element creates a dropdown menu. It uses <option> elements for the available choices.
The <button> element is used for actions like form submission or triggering JavaScript events.
